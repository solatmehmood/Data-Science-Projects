# 🧠 Data Science Projects Portfolio

Welcome to my portfolio of Data Science projects. This repository is divided into two main sections:

## 📊 Data Analysis Projects
1. **Daily Life Habits vs Productivity** — Analyzing lifestyle habits and their impact on productivity.
2. **Climate Change Trends** — Exploring global climate indicators over decades.
3. **Traffic Accidents EDA** — Examining patterns in urban traffic accidents.
4. **Netflix Content Analysis** — Visualizing trends in Netflix content and genres.

## 🤖 Machine Learning Projects
1. **Student Performance Prediction** — Predicting student scores based on lifestyle.
2. **Diabetes Prediction** — Classification using medical data.
3. **House Price Predictor** — Regression model to estimate house prices.
4. **Customer Churn Prediction** — ML model to predict if a customer will leave.

Each folder contains:
- 📁 Jupyter Notebook (`.ipynb`)
- 📊 Dataset
- 📄 README explaining the project

> **Tools Used**: Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn
